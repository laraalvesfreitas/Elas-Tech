# Elas + Tech 
Repositório com conteúdos, exercícios e projetos desenvolvidos durante o programa Elas + Tech, com foco na formação em desenvolvimento web e aprimoramento de habilidades em programação.

Ao longo da trajetória, foram estudados e aplicados conceitos fundamentais e avançados de desenvolvimento, incluindo:

- JavaScript
- JSON
- Front-end Dinâmico
- HTML5
- CSS3
- React
- TypeScript

O repositório reúne materiais de estudo, desafios práticos, exercícios e projetos desenvolvidos para consolidar conhecimentos em lógica de programação, manipulação de dados, desenvolvimento de interfaces responsivas e construção de aplicações web modernas, seguindo boas práticas de desenvolvimento de software.
